README

Пакет, предоставляющий набор инструментов для создания и управления скоупами на
Flutter. В том числе для внедрения зависимостей, асинхронной инициализации и
асинхронной утилизации.

Использование сторонних пакетов это торчащие наружу элементы технической
реализации (например, `Provider.of<User>(context)`, `myBloc` и т.п.).
Использование же родных инструментов Flutter позволяет скрыть техническую
реализацию и оставить только бизнес логику (`User.of(context)`).
